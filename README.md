# 🏏 IPL Data Analysis Project (Batting + Bowling)

## 📌 Overview

This project is a comprehensive data analysis of IPL 2025 player performance, covering both **batting** and **bowling** aspects. The goal is to extract meaningful insights and identify the key characteristics of top-performing players using Python and data analysis libraries.

---

## 🚀 Objectives

* Analyze batting performance metrics
* Analyze bowling performance metrics
* Identify patterns of top players
* Build a strong foundation for future Machine Learning models

---

## 📂 Project Structure

```
IPL_Project/
│
├── Batting_Analysis.ipynb
├── Bowling_Analysis.ipynb
├── datasets/
│   ├── batting_data.csv
│   └── bowling_data.csv
├── README.md
```

## 📊 Batting Analysis

### 🔍 Key Metrics Used

* Total Runs
* Batting Average
* Strike Rate
* Boundaries (4s & 6s)
* Consistency

### 📈 Insights Extracted

* Identified top run scorers
* Players with highest strike rates
* Consistency vs aggression comparison
* Ideal traits of a good T20 batter

---

## 🎯 Bowling Analysis

### 🔍 Key Metrics Used

* Wickets Taken
* Economy Rate
* Bowling Average
* Strike Rate (balls per wicket)

### 📈 Insights Extracted

* Most effective bowlers
* Economy vs wicket-taking ability
* Key traits of a successful T20 bowler
